# Paint-but-code
A C++ SFML-based drawing tool that converts shapes drawn on the window into reusable SFML code.
## ✨ Features

- 🎨 **Customizable Color Palette**
  - Choose from a wide range of colors for drawing and designing

- 🔷 **18 Different Shapes**
  - Draw multiple predefined shapes directly on the canvas

- 🪣 **Color Fill Option**
  - Fill shapes with your selected color for better visual customization

- ✏️ **Pencil Tool**
  - Freehand drawing support for more flexible sketching

- 🌈 **Blending Tools**
  - Blend and experiment with colors for smoother design creation

- 🖌️ **Adjustable Stroke Size**
  - Change brush and outline thickness based on your needs

- 💾 **Picture Save Option**
  - Save your artwork as an image for future use

- 🖥️ **User-Friendly Interface**
  - Designed to be simple, intuitive, and easy to use

- ⚙️ **SFML Code Generation**
  - Convert drawn content into reusable SFML code

---

## 🚀 Purpose

This application is designed to make graphical design in **SFML** easier and more interactive.

Instead of manually writing code for every visual element, users can draw directly on the screen and use the generated code as a starting point for their own projects.

It can be useful for:
- Students working on graphics-based projects
- Developers who want to prototype 2D visuals more efficiently

---

## 🛠️ Technologies Used

- **C++**
- **SFML (Simple and Fast Multimedia Library)**

##How to install 
- Install the .zip file for your OS. I have made this tool for linux and windows
- Simply run the sfml_paint_pro on linux and sfml_paint_pro.exe on windows
